### Features Implemented

**Focus Trap (WCAG 2.1.2)**
- Activates on `[role="dialog"][aria-modal="true"]` insertion via `MutationObserver`
- Deactivates on removal; restores focus to `data-focus-return` element
- Uses `AbortController` + `{ signal }` for scoped keydown listener teardown
- Single module-level `activeTrapController` (Phase 1: no nested modal support)
- Handles empty-modal edge case: falls back to focusing dialog root

**Table Grid Navigation (WCAG 2.1.1 / 4.1.3)**
- Single `document.addEventListener('keydown', ..., { capture: true })` delegation
- ArrowUp / ArrowDown navigate between `tbody tr[tabindex="0"]` rows
- Home / End jump to first / last row
- Event propagation stopped after handling

### Key Decisions
- `AbortController` teardown over stored listener references (ES2024+ idiomatic)
- Load as `defer` classic script, not `type="module"` (no exports, side-effect only)
- `MutationObserver` on `document.body`, not `document` (avoids `<head>` noise)
- Focusable filter via `.filter(el => !el.closest('[hidden]') && !el.closest('[inert]'))` (runtime filter, not CSS selector embedding)
